# Screens

## Akış
```
HomeScreen → IntervalConfigScreen → IntervalQuizScreen
          → IntervalLearnScreen
```

## Dosyalar
- `HomeScreen.tsx` - Ana menü, 2 buton (Test, Learn)
- `IntervalConfigScreen.tsx` - Quiz ayarları (interval seçimi, direction, sustain)
- `IntervalQuizScreen.tsx` - Quiz modu, skor takibi, melodic/harmonic playback
- `IntervalLearnScreen.tsx` - Öğrenme modu, interval seçip dinleme, lock base note
